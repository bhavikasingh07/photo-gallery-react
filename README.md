# Photo Gallery React App

A responsive photo gallery built using React and Vite. The app fetches photos from a public API and allows users to search photos by author and mark their favorite images.

## Features
- Fetch photos from API
- Real-time search filter by author name
- Add or remove photos from favorites
- Favorites saved in localStorage
- Responsive grid layout
- Performance optimization using React hooks (useMemo, useCallback, useReducer)

## Tech Stack
- React
- Vite
- JavaScript (ES6)
- Tailwind CSS

## How to Run

Clone the repository:
git clone https://github.com/bhavikasingh07/photo-gallery-react.git

Install dependencies:
npm install

Run the project:
npm run dev

Open in browser:
http://localhost:5173
